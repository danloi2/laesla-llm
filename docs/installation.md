# Installation

## Stable release

To install Python Lates, run this command in your terminal:

```sh
uv add python-lates
```

Or if you prefer to use `pip`:

```sh
pip install python-lates
```

## From source

The source files for Python Lates can be downloaded from the [Github repo](https://github.com/danloi2/python_lates).

You can either clone the public repository:

```sh
git clone git://github.com/danloi2/python_lates
```

Or download the [tarball](https://github.com/danloi2/python_lates/tarball/master):

```sh
curl -OJL https://github.com/danloi2/python_lates/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd python_lates
uv pip install .
```
