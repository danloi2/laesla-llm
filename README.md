## Usage

To run the Latin to Spanish translator interactively:

```bash
python_lates translate
# or
python3 -m python_lates translate
```

To regenerate the vocabulary file from the model:

```bash
python_lates fix-vocab
# or
python3 -m python_lates fix-vocab
```
